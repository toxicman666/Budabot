<header>::::: Planters Commands :::::<end>

<highlight>Shows available planters<end>
<tab><symbol>planters

<highlight>Shows available planters for a site<end>
<tab><symbol>planters 'zone' 'number'

<highlight>Displays control tower types by level<end>
<tab><symbol>towertypes

<highlight>Add yourself (and all alts) to planters<end>
<tab><symbol>iamplanter

<highlight>Remove yourself (and all alts) from planters<end>
<tab><symbol>iamplanter not

<highlight>Finish lookup if it's stuck for some reason<end>
<tab><symbol>planters end

<highlight>Remove planter from the list<end>
<tab><symbol>remplanter 'name'
